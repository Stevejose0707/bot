RAG-Based Customer Support Chatbot with Vector SearchInteractive AI chatbot for BRAND'S® Essence of Chicken using Streamlit, Ollama LLM, and FAISS vector database. Features: multi-stage conversation flow (greeting→registration→chat), PDF knowledge base ingestion with sentence transformers, semantic search for context-aware responses, CRM integration with code validation, user registration with lucky draw enrollment. Built with Python, SentenceTransformers, PyPDF2. Supports local LLM deployment via Ollama for data privacy.
